# aplikasi-penentuan-nilai-tempat-bilangan
Aplikasi Python sederhana untuk menentukan nilai tempat setiap digit pada sebuah bilangan (satuan, puluhan, ratusan, ribuan, dan seterusnya). Program menerima input berupa angka, lalu menampilkan posisi masing-masing digit menggunakan konsep pembalikan string dan perulangan. Cocok untuk pembelajaran dasar Python dan pemahaman nilai tempat bilangan.
